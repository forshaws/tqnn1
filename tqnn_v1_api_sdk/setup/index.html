Setup authID API

You are currently running in demo mode

This repository comes with 2 example applications that will help you quickly get started with TORIDION™ authID API. You can begin using the example apps right now using a demo APIKEY, but your data will not be saved.

Try the examples?

To try the samples without setting up an APIKEY click "Examples" in the top menu. These examples use a demo APIKEY and APISECRET and will let you actually register and validate credentials on a live TORIDION™ authID API instance.

Remember in demo mode your data wont be saved for long so do not use this for live environment or send sensitive data.

Get a key

To create an APIKEY and APISECRET that will allow you to take advantage of the TORIDION™ authID API click "Get Key" in the top menu.

Already have a key?

If you already have a valid APIKEY and APISECRET and you wish to link this application tot he same key you can enter your credentials in the form below. This will generate your configuration files and terminate the setup wizard.
